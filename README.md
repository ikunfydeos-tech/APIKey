## 常见问题

### Q: 密钥安全吗？

**A:** 我们采用 AES-256 加密存储所有密钥，密钥永不明文显示。即使是管理员也无法查看你的密钥原文。

### Q: 你们会使用我的 API 密钥吗？

**A:** 不会。我们只负责安全存储，不会调用你的密钥。测试连接功能由你的浏览器直接请求服务商 API。

### Q: 免费版够用吗？

**A:** 如果你管理不超过 10 个密钥，免费版完全够用。需要更多密钥或高级功能可升级 Pro 版。

### Q: 如何取消订阅？

**A:** 随时可在账户设置中取消订阅，已付费的服务期将继续有效。

### Q: 数据会丢失吗？

**A:** Pro 版提供每日自动备份，即使发生意外也能恢复数据。
